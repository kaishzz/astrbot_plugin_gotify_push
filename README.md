# astrbot_plugin_gotify_push

### 介绍

- 监听 Gotify 消息并推送

---

| 参数         | 解释                                         |
| ------------ | -------------------------------------------- |
| Server       | 请输入服务器地址                             |
| Client Token | Gotify Client Token                          |
| Application  | 要监听的 Gotify Application, 留空则监听全部. |
| UMO          | 推送的会话, 可以推送多个, 值为 /sid 的 UMO.  |
